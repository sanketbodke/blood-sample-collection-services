# README

* Ruby version => 3.2.1
* Rails version => 7.2.2

To install all dependencies:
```
bundle install
```

Create Database:
```
rails db:create
```

Drop Database:
```
rails db:drop
```

Migrate Database:
```
rails db:migrate
```

To start server:
```
rails s
```

To open console:
```
rails c
```
